# 3D Printer Queue Scheduler

A modern web application for managing and scheduling 3D printer jobs. Built with Next.js, React, and TypeScript, featuring drag-and-drop queue management for efficient print job organization.

## Live Demo

This project is available at [queue.bnap.dev](https://queue.bnap.dev)

## Features

- Real-time print queue management
- Drag-and-drop interface for reordering print jobs
- Current print status tracking
- Add and manage print jobs with ease
- Responsive design with Tailwind CSS

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- [Next.js 16](https://nextjs.org) - React framework
- [React 19](https://react.dev) - UI library
- [TypeScript](https://www.typescriptlang.org) - Type safety
- [Tailwind CSS](https://tailwindcss.com) - Styling
- [dnd-kit](https://dndkit.com) - Drag and drop functionality
- [Biome](https://biomejs.dev) - Linting and formatting
